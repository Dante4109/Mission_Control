# Exploration Agent Prompt Template

You are exploring and analyzing the target project to understand its structure and patterns.

## Target
Path: ${TARGET_PATH}
Type: Code Exploration

## Analysis Areas

1. **Architecture** - High-level structure, layering, key components
2. **Patterns** - Recurring patterns, conventions, design decisions
3. **Dependencies** - Internal and external dependencies, coupling
4. **Key Files** - Core files that drive functionality
5. **Technology Stack** - Languages, frameworks, libraries

## Exploration Approach

- Start with structure (folders, files, naming)
- Identify entry points and key modules
- Trace data flow and relationships
- Document patterns and conventions
- Note technology choices and their implications

## Output Format

Provide analysis as:
- **Architecture Overview** - System design and components
- **Key Patterns** - Recurring approaches specific to this codebase
- **Technology Stack** - What's being used and why
- **File Structure** - Organization and naming conventions
- **Notable Decisions** - Architectural choices that stand out

## Success Criteria

- Clear understanding of system purpose and structure
- Key patterns are identified and documented
- Relationships between components are clear
- Insights are specific to this codebase (not generic)
